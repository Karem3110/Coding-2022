The main languages of the web are HTML, CSS and JavaScript.


HTML

What is HTML?

HTML stands for Hypertex Markup Language and it is a computer language for creating most web pages and online applications. HTML describes the structure of a web page, and consists of a series of elements. HTML elements tell the browser how to display the content.

Anatomy of an HTML elements

The main parts of an element are the followings:

  1. The opening tag
  2. The closing tag
  3. The content
  4. The element

CSS

What is CSS?

CSS stands for Cascading Style Sheets, and it is a language that describes how HTML elements are to be displayed on screen, paper, or in other media. CSS saves a lot of work, because it can control the layout of multiple web pages all at once.

CSS controls the presentation of a webpage and this includes:

  • Layout.
  • Colour of text and backgrounds.
  • Size of HTML elements.
  • Fonts.

CSS consists of selectors, properties and values.

Anatomy of an CSS ruleset

The whole structure is called a ruleset. (The term ruleset is often referred to as just rule.) Note the names of the individual parts:

  • Selector.-This is the HTML element name at the start of the ruleset. It defines the element(s) to be styled (in this example,</p> elements). To style a different element, change the selector.
  • Declaration.- This is a single rule like color: red;. It specifies which of the element's properties you want to style.
  • Properties.- These are ways in which you can style an HTML element. (In this example, color is a property of the </p> elements.) In CSS, you choose which properties you want to affect in the rule.
  • Property value.- To the right of the property—after the colon—there is the property value. This chooses one out of many possible appearances for a given property. (For example, there are many color values in addition to red.)
